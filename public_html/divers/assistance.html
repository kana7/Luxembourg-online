Configuration E-mail

Séléctionnez la configuration:

Paramètres de base

  • Serveur de courrier entrant (POP) : pop.internet.lu
  • Port: 110/connexion sans SSL
  • Serveur de courrier sortant (SMTP) : smtp.internet.lu
  • Port SMTP: 25 ou 587 avec authentification activée/connexion sans SSL
  • Adresse email : votre adresse @internet.lu
  • Nom d'utilisateur (username) : votre username
  • Mot de passe (password) : votre password

Windows Mail

  • Cliquez dans la barre de menu à droite sur Paramètres.
  • Cliquez sur Comptes, puis sur Ajouter un compte.
  • Sélectionnez Autre compte.
  • Cochez la case POP et cliquez ensuite sur Connecter.
  • Cliquez sur Afficher plus de détails et remplissez les champs comme indiqué ci-dessous.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Nom d'utilisateur : réintroduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser ce mot de passe".
    • Serveur de messagerie entrant (POP) : introduisez pop.internet.lu dans ce champ.
    • Port : 110
    • Le serveur entrant requiert SSL : NE PAS cocher cette case.
    • Serveur de messagerie sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Port : 25 ou 587
    • Le serveur sortant requiert SSL : NE PAS cocher cette case.
    • Le serveur sortant requiert l’authentification : cochez cette case.
    • Utiliser les mêmes nom d’utilisateur et mot de passe pour envoyer et recevoir les messages électroniques : cochez cette case.
  • Cliquez sur Connecter.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Windows Mail.

Windows Live Mail

  • Dans la barre de menu, cliquez sur Comptes et ensuite sur Ajouter.
  • Cliquez sur Compte de messagerie et ensuite sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur Suivant.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser ce mot de passe".
    • Nom d’affichage : introduisez votre nom. Vos contacts verront apparaître ce nom lorsqu’ils recevront vos e-mails.
    • Configurer manuellement les paramètres du serveur : cochez cette case.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur Suivant.
      • Informations sur le serveur entrant
    • Type de serveur : sélectionnez POP dans le menu déroulant.
    • Adresse du serveur : introduisez pop.internet.lu dans ce champ.
    • Port : 110
    • Connexion sécurisée (SSL) requise : NE PAS cocher cette case.
    • Identifiant de connexion : introduisez votre nom d'utilisateur.
      Informations sur le serveur sortant
    • Adresse du serveur : introduisez smtp.internet.lu dans ce champ.
    • Port : 25 ou 587
    • Connexion sécurisée (SSL) requise : NE PAS cocher cette case.
    • Authentification requise : cochez cette case.
  • Cliquez sur Terminer, ensuite sur Arrêter.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Windows Live Mail.

Outlook 2013

  • Dans la barre de menu, cliquez sur Fichier et ensuite sur Informations.
  • Cliquez sur le bouton Ajouter un compte.
  • Cochez la case "Configuration manuelle ou types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "POP ou IMAP" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre prénom et votre nom de famille.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser le mot de passe".
  • Allez dans l'onglet Options avancées, remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton OK
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucun.
    • Serveur entrant (POP) : introduisez 110.
    • Serveur de courrier sortant (SMTP) : introduisez 25 ou 587.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucun.
  • Cliquez sur Suivant, puis sur Fermer et ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook 2013.

Outlook 2010

  • Dans la barre de menu, cliquez sur Fichier et ensuite sur Informations.
  • Cliquez sur le bouton Ajouter un compte.
  • Cochez la case "Configurer manuellement les paramètres du serveur ou les types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "Messagerie Internet" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre nom.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser le mot de passe".
  • Allez dans l'onglet Options avancées, remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton OK.
    • Serveur entrant (POP) : introduisez 110.
    • Ce serveur nécessite une connexion chiffrée (SSL) : NE PAS coher cette case.
    • Serveur sortant (SMTP) : introduisez 25 ou 587.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucun
  • Cliquez sur Suivant, ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook 2010.

Outlook 2007

  • Dans la barre de menu, cliquez sur Outils et ensuite sur Paramètres du compte.
  • Allez dans l'onglet Messagerie et cliquez sur Nouveau.
  • Cochez la case "Microsoft Exchange, POP3, IMAP ou HTTP" et cliquez sur Suivant.
  • Cochez la case "Configurer manuellement les paramètres du serveur ou les types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "Messagerie Internet" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre nom.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP3 dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d’utilisateur : réintroduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case " Mémoriser le mot de passe".
  • Allez dans l’onglet Options avancées, complétez les champs comme décrit ci-dessous et cliquez sur le bouton OK.
    • Serveur entrant (IMAP) : introduisez 110.
    • Ce serveur nécessite une connexion chiffrée (SSL) : NE PAS cocher cette case
    • Serveur sortant (SMTP) : introduisez 25 ou 587.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucune.
  • Cliquez sur Suivant, ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook 2007.

Apple Mail

  • Dans la barre de menu, cliquez sur Mail et ensuite sur Préférences.
  • Sélectionnez l’onglet Comptes et cliquez sur le bouton "+".
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton Continuer.
    • Nom complet : introduisez votre nom.
    • Adresse e-mail: ne complétez que la partie de votre adresse e-mail se trouvant avant le symbole @.
    • Mot de passe : introduisez votre mot de passe.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton Continuer.
    • Type de compte : sélectionnez POP.
    • Description : si vous le souhaitez, vous pouvez donner un nom à votre compte.
    • Serveur de réception : introduisez pop.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre nom d'utilisateur.
    • Mot de passe : si ce champ n'a pas été rempli automatiquement, réintroduisez ici votre mot de passe.
  • Authentification : Choisir Mot de passe.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton Continuer.
    • Description : si vous le souhaitez, vous pouvez donner un nom à votre compte.
    • Serveur d’envoi : introduisez smtp.internet.lu dans ce champ.
    • N'utiliser que ce serveur : cochez cette case.
    • Nom d'utilisateur : ne remplissez pas ce champ.
    • Mot de passe : ne remplissez pas ce champ.
  • Authentification : Choisir Mot de passe.
  • Cliquez sur Créer.
  • Sélectionnez votre compte dans l’onglet Comptes.
  • Remplissez les champs comme indiqué ci-dessous.
    • Adresse électronique : introduisez votre adresse e-mail.
    • Serveur d’envoi (SMTP) : sélectionnez Modifier la liste des serveurs SMTP dans le menu déroulant.
  • Sélectionnez l’onglet Avancé, remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton OK.
    • Utiliser le port personnalisé : introduisez 25 ou 587.
    • Utiliser SSL : NE PAS cocher cette case.
    • Authentification : sélectionnez Mot de passe dans le menu déroulant.
    • Nom d’utilisateur : introduisez votre adresse e-mail complète.
    • Wachtwoord : introduisez votre mot de passe.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Apple Mail.
