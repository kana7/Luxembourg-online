Configuration mail

Séléctionnez la configuration :

Paramètres de base

  • Serveur de courrier entrant (POP) : pop.internet.lu
  • Port: 110/connexion sans SSL
  • Serveur de courrier sortant (SMTP) : smtp.internet.lu
  • Port SMTP: 25 ou 587 avec authentification activée/connexion sans SSL
  • Adresse email : votre adresse @internet.lu
  • Nom d'utilisateur (username) : votre username
  • Mot de passe (password) : votre password

Windows 10 Mail

  • Ouvrez l'application Mail (Courrier en francais), cliquez sur l'icone "Paramètres" puis "Comptes".
  • Cliquez ensuite sur "Ajouter un compte".
  • Sélectionnez "Autre compte".
  • Entrez l’adresse email et le mot de passe correspondant et selectionnez "Ouverture de session".
  • Cliquez sur "Terminé".

Une fenêtre de confirmation apparait : votre compte est configuré ! Vous pouvez maintenant envoyer et recevoir vos emails via Windows Mail 10.

Pour plus d'information : Assistance Microsoft

Windows 8 Mail

L'application Windows 8 Mail n'est malheureusement pas compatible car elle ne supporte pas le protocole POP3 avec le service Mail Luxembourg Online.
Pour consulter vos emails, veuillez utiliser un autre client mail (Thunderbird, Outlook, ...) ou accéder à l'espace client du site www.internet.lu.

Outlook (valable pour Outlook 2010,2013 et 2016)

  • Démarrez Outlook.
  • Dans la barre de menu, cliquez sur Fichier et ensuite sur Ajouter un compte.
  • Cochez la case "Configuration manuelle ou types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "POP ou IMAP" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre prénom et votre nom de famille.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser le mot de passe".
  • Cliquez ensuite sur "Paramètres supplémentaires ..."
    • Allez dans l'onglet Options avancées, remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton OK.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucun.
    • Serveur entrant (POP) : introduisez 110.
    • Serveur de courrier sortant (SMTP) : introduisez 25 ou 587.
  • Cliquez sur Suivant, puis sur Fermer et ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook.

Pour plus d'information : Assistance Microsoft

Outlook 2007

  • Dans la barre de menu, cliquez sur Outils et ensuite sur Paramètres du compte.
  • Allez dans l'onglet Messagerie et cliquez sur Nouveau.
  • Cochez la case "Microsoft Exchange, POP3, IMAP ou HTTP" et cliquez sur Suivant.
  • Cochez la case "Configurer manuellement les paramètres du serveur ou les types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "Messagerie Internet" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre nom.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP3 dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d’utilisateur : réintroduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case " Mémoriser le mot de passe".
  • Allez dans l’onglet Options avancées, complétez les champs comme décrit ci-dessous et cliquez sur le bouton OK.
    • Serveur entrant (POP) : introduisez 110.
    • Ce serveur nécessite une connexion chiffrée (SSL) : NE PAS cocher cette case
    • Serveur sortant (SMTP) : introduisez 25 ou 587.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucune.
  • Cliquez sur Suivant, ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook 2007.

Activer les données mobiles

  • Appuyez sur le lanceur d'applications.
  • Sélectionnez "Paramètres" ou "Réglages".
  • En fonction de la marque et de la version Android de votre mobile, sélectionnez :
    • "Sans fil et réseaux"
    • ou "Plus..."
    • ou "Données mobiles"
    • ou "Paramètres supplémentaires"
    • ou "Plus de réseaux" (depuis l'onglet "Connexions")
    • ou "Réseaux et partage de connexion"(depuis l'onglet "Réseaux").
  • Sélectionnez "Réseaux mobiles".
  • Assurez-vous que la première ligne "Données mobiles", "Données mobiles activées", "Données activées" ou "Trafic de données mobiles" est bien cochée.

Activer l'itinérance des données (roaming)

  • Rendez-vous dans "Paramètres" > "Réseaux Mobiles".
  • La case "Données mobiles" ou "Itinérance" doit être cochée.

Si vous cochez cette case, un message d'information à propos vous informera des risques de facturation de la data à l’étranger.

Configuration des paramètres APN

L'APN (Access Point Name) est le point d’accès qu'utilise votre téléphone pour accéder au réseau Internet mobile de votre opérateur.

Etape 1 : accédez aux paramètres APN

  • Appuyez sur le lanceur d'applications.
  • Sélectionnez "Paramètres" ou "Réglages".
  • Suivant la marque et la version Android de votre mobile, sélectionnez :
    • "Sans fil et réseaux"
    • ou "Plus..."
    • ou "Données mobiles"
    • ou "Paramètres supplémentaires"
    • ou "Plus de réseaux" (depuis l'onglet "Connexions")
    • ou "Réseaux et partage de connexion"(depuis l'onglet "Réseaux").
  • Sélectionnez "Réseaux mobiles".
  • Sélectionnez ensuite "Nom des points d'accès".

Plusieurs cas sont possibles lorsque vous êtes sur la page de configuration des paramètres APN :

- soit vous pouvez modifier les APN existants en appuyant simplement dessus et en renseignant les informations ci-après,

- soit vous enregistrez de nouveaux APN et supprimez les anciens APN déjà paramétrés.

Etape 2 : configurer les paramètres APN LOL

Renseignez les informations présentes dans la configuration Paramètre de base.

En cas de difficultés, faites appel à notre service Helpdesk.

Configuration des paramètres APN

L'APN (Access Point Name) est le point d’accès qu'utilise votre téléphone pour accéder au réseau Internet mobile de votre opérateur.


Etape 1 : accédez aux paramètres APN

  • Depuis l'écran d'accueil, appuyez sur la flèche en bas à droite de l'écran (ou faites défiler l'écran vers la gauche) pour accéder à la liste des applications.
  • Sélectionnez "Paramètres".
  • Sélectionnez ensuite "réseau cellulaire" ou "réseau mobile".
  • En fonction de la version de Windows de votre mobile, appuyez sur "ajouter un APN internet" (ou "ajouter un APN mms", suivant l'APN que vous souhaitez renseigner) ou sélectionnez "modifier APN".

Plusieurs cas sont possibles lorsque vous êtes sur la page de configuration des paramètres APN :

- soit vous pouvez modifier les APN existants en appuyant simplement dessus et en renseignant les informations ci-après,

- soit vous enregistrez de nouveaux APN et supprimez les anciens APN déjà paramétrés.

Etape 2 : configurer les paramètres APN LOL

Renseignez les informations présentes dans la configuration Paramètre de base.

Vous avez encore des questions ? Contactez-nous !

Quels sont les avantages de la domiciliation ?

La domiciliation est un mode de paiement simple et sécurisé. Le prélèvement est effectué automatiquement tous les mois, évitant ainsi les frais de facturation (1,50 €) et les frais en cas de retard de paiement. Pour mettre en place une domiciliation, lisez la question suivante.

Comment mettre en place une domiciliation ?

Si vous souhaitez que votre facture soit directement prélevée sur votre compte bancaire, vous devez :

  • imprimer et remplir le mandat de domiciliation,
  • nous le faire parvenir en ligne via notre formulaire, par voie postale, ou l'apporter en boutique.

Quelle est la date de prélèvement ?

Dès lors que votre domiciliation est active, les prélèvements se font en fin de mois ou bien le 15 du mois, si la première tentative a échoué.

Peut-on changer la date de prélèvement ?

A ce jour, il n'est pas possible de changer la date de prélèvement. Toutefois, si vous rencontrez des difficultés, Contactez-nous.

Comment payer ma facture en ligne ?

Pour payer votre facture en ligne, suivez les étapes suivantes :

  • Cliquez sur Espace client, puis séléctionez mon compte.
  • Connectez-vous à l'aide de votre identifiant et de votre mot de passe.
  • Cliquez sur « Compte client » puis sur le bouton « Payer XX EUR » ( )
  • Sélectionnez ensuite le lien correspondant à votre mode de paiement.
  • Suivez les instructions indiquées par votre banque. A la fin du processus, vous recevrez une confirmation de paiement à conserver.

Quels sont les avantages de la facture par mail ?

  • Vous n'avez plus à payer les frais d'envoi (1.50 euro par mois).
  • Vous pouvez consulter vos factures partout et à tout moment, en accédant à vos mails ou à votre Espace client.
  • Vous archivez plus facilement vos factures, en les enregistrant directement sur votre ordinateur, ou même sur votre LOLCLoud !
  • Vous ne recevez plus de facture papier : ainsi, vous contribuez à la protection de l'environnement.

Pour recevoir votre facture par email, il vous suffit d'en faire la demande ici.

Comment recevoir ma facture par mail ?

Pour recevoir ma facture par mail, il suffit d'en faire la demande ici.

Comment consulter ma facture en ligne ?

Pour cela, il vous suffit de vous connectez à votre Espace client, cliquez sur "Mon compte" et entrez ensuite vos identifiants dans les champs correspondants. Cliquez sur "se connecter", puis sélectionnez « Compte client ». Vous pourrez visualiser toutes vos factures et également payer en ligne.

Comment puis-je suivre ma consommation ?

  • Cliquez sur Espace client, puis séléctionez mon compte.
  • Connectez-vous à l'aide de votre identifiant et de votre mot de passe.
  • Cliquez sur « Mes abonnements » puis sur « Consommation ».

Vous pouvez visualiser ici le détail de vos consommations.

Je constate une erreur sur ma facture, que faire ?

Si vous estimez que votre facture comporte une erreur, nous vous invitons à nous contacter en utilisant le formulaire.
