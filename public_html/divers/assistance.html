Configuration E-mail

Séléctionnez la configuration:

Paramètres de base

  • Serveur de courrier entrant (POP) : pop.internet.lu
  • Port: 110/connexion sans SSL
  • Serveur de courrier sortant (SMTP) : smtp.internet.lu
  • Port SMTP: 25 ou 587 avec authentification activée/connexion sans SSL
  • Adresse email : votre adresse @internet.lu
  • Nom d'utilisateur (username) : votre username
  • Mot de passe (password) : votre password

Windows 10 Mail

  • Ouvrez l'application Mail (Courrier en francais), cliquez sur l'icone "Paramètres" puis "Comptes".
  • Cliquez ensuite sur "Ajouter un compte".
  • Sélectionnez "Autre compte".
  • Entrez l’adresse email et le mot de passe correspondant et selectionnez "Ouverture de session".
  • Cliquez sur "Terminé".

Une fenêtre de confirmation apparait : votre compte est configuré ! Vous pouvez maintenant envoyer et recevoir vos emails via Windows Mail 10.

Pour plus d'information : Assistance Microsoft

Windows 8 Mail

L'application Windows 8 Mail n'est malheureusement pas compatible car elle ne supporte pas le protocole POP3 avec le service Mail Luxembourg Online.
Pour consulter vos emails, veuillez utiliser un autre client mail (Thunderbird, Outlook, ...) ou accéder à l'espace client du site www.internet.lu.

Outlook (valable pour Outlook 2010,2013 et 2016)

  • Démarrez Outlook.
  • Dans la barre de menu, cliquez sur Fichier et ensuite sur Ajouter un compte.
  • Cochez la case "Configuration manuelle ou types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "POP ou IMAP" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre prénom et votre nom de famille.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case "Mémoriser le mot de passe".
  • Cliquez ensuite sur "Paramètres supplémentaires ..."
    • Allez dans l'onglet Options avancées, remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton OK.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucun.
    • Serveur entrant (POP) : introduisez 110.
    • Serveur de courrier sortant (SMTP) : introduisez 25 ou 587.
  • Cliquez sur Suivant, puis sur Fermer et ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook.

Pour plus d'information : Assistance Microsoft

Outlook 2007

  • Dans la barre de menu, cliquez sur Outils et ensuite sur Paramètres du compte.
  • Allez dans l'onglet Messagerie et cliquez sur Nouveau.
  • Cochez la case "Microsoft Exchange, POP3, IMAP ou HTTP" et cliquez sur Suivant.
  • Cochez la case "Configurer manuellement les paramètres du serveur ou les types de serveurs supplémentaires" et cliquez sur Suivant.
  • Cochez la case "Messagerie Internet" et cliquez sur Suivant.
  • Remplissez les champs comme indiqué ci-dessous et cliquez sur le bouton "Paramètres supplémentaires".
    • Votre nom : introduisez votre nom.
    • Adresse de messagerie : introduisez votre adresse e-mail.
    • Type de compte : sélectionnez POP3 dans le menu déroulant.
    • Serveur de courrier entrant : introduisez pop.internet.lu dans ce champ.
    • Serveur de courrier sortant (SMTP) : introduisez smtp.internet.lu dans ce champ.
    • Nom d’utilisateur : réintroduisez votre nom d'utilisateur.
    • Mot de passe : introduisez votre mot de passe. Si vous le souhaitez, vous pouvez cocher la case " Mémoriser le mot de passe".
  • Allez dans l’onglet Options avancées, complétez les champs comme décrit ci-dessous et cliquez sur le bouton OK.
    • Serveur entrant (IMAP) : introduisez 110.
    • Ce serveur nécessite une connexion chiffrée (SSL) : NE PAS cocher cette case
    • Serveur sortant (SMTP) : introduisez 25 ou 587.
    • Utilisez le type de connexion chiffrée suivant : sélectionnez aucune.
  • Cliquez sur Suivant, ensuite sur Terminer.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Outlook 2007.

Apple Mail

  • Ouvrir l'application Mail. S'il s'agit de la première fois, vous serez directement invité à ajouter un compte. Si ce n'est pas le cas, sélectionner l'onglet Mail, puis Ajouter un compte.
  • La fenêtre "Choisir un compte mail à ajouter..." s'affiche. Sélectionnez "Ajouter un autre compte mail..." et cliquez sur Continuer.
  • Entrez vos informations comme indiqué ci-dessous puis cliquez sur Continuer
    • Nom complet : introduisez votre nom tel que vous souhaitez qu'il apparaisse.
    • Adresse e-mail : introduisez votre adresse e-mail complète.
    • Mot de passe: introduisez le mot de passe associé à votre compte de messagerie.
  • Mot de passe: introduisez le mot de passe associé à votre compte de messagerie.
    • Type de compte : sélectionnez POP.
    • Description : vous pouvez donner un nom à votre compte (facultatif)
    • Serveur de réception : introduisez pop.internet.lu dans ce champ.
    • Nom d'utilisateur : introduisez votre adresse e-mail
    • Mot de passe : si ce champ n'a pas été rempli automatiquement, réintroduisez ici votre mot de passe.
  • A la fin, selectionner à nouveau Continuer
  • Des informations supplémentaires vous sont demandées. Saisissez-les comme indiqué ci-dessous.
    • Path prefix : laisser ce champ vide.
  • Vous pouvez désormais envoyer et recevoir vos e-mails via Apple Mail.

Pour pour d'information : Assistance mail Apple
