Le plus souvent, votre téléphone mobile récupère automatiquement les paramètres de configuration Internet depuis la SIM.
Si la data ne fonctionne pas, vérifiez les points suivants :

  • - les données mobiles sont-elles bien activées ?
  • - l’itinérance des données (roaming) est-elle activée ?*
  • - les paramètres APN sont-ils correctement configurés ?
  • - avez-vous testé votre carte SIM avec un autre téléphone mobile ?
  • - avez-vous du réseau ?

À noter

Vous avez effectué tous les tests mais le problème persiste ? Contactez-nous.

*L’activation de l’itinérance des données entraine des couts supplémentaires. Tarification détaillée disponible sur internet.lu, rubrique Mobile.
